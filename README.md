
# Character Counter

## Description

This project is a simple **Character Counter** built using **HTML, CSS, and JavaScript**. It allows users to enter text in a textarea and displays the number of characters typed in real time. The application also shows the remaining characters available based on a predefined limit.

The project uses HTML to create the user interface, CSS to improve the appearance and layout, and JavaScript to handle user input and update the character count dynamically. When the user reaches the maximum character limit, a warning message is displayed and additional input is prevented.

## Features

* Real-time character counting
* Displays typed and remaining characters
* Maximum character limit support
* Warning message when the limit is reached
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose

The main purpose of this project is to demonstrate DOM manipulation, event handling, and form input validation using JavaScript while creating an interactive and responsive user experience.
